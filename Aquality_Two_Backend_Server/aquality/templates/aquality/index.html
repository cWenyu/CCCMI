What is Aquality

Aquality 1.0 is an application that helps determine water quality through the use of Insects.

Aquality 2.0 is an improvement of version 1.0. It helps determine water quality through the use of 3 sensors (PH, temperature and oxygen content) and the AI model to analysis insects to enhance the accuracy of data which is collected by Citizen Scientists.

The whole project applies IoT technic, combines hardware and software together to achieve the purpose of automation.

Technology

  • Hardware: ESP 32 Microcontroller with PH sensor and temperature sensor
  • App: React Native
  • Cloud: AWS, PubNub
  • AI: Yolov3+LabelImg
  • Backend: Django + PostgreSQL
  • UI Testing: Appium + WendriveIO
  • API Testing: Postman + newman + Github Actions
